# 🧴 화해 모티브 화장품 사이트 (Final Project)

## 📌 프로젝트 개요
이 프로젝트는 **화해를 모티브**로 한 화장품 사이트로, 고객 응대를 중심으로 기능을 설계하고 개발했습니다.  
메인 화면 및 전반적인 **UI/UX 디자인** 또한 직접 담당하여 사용자 친화적인 화면 구성을 구현했습니다.  

제가 담당한 주요 기능은 다음과 같습니다:
- 게시판 (공지사항, 문의사항, 이벤트 등)
- 대댓글 기능 지원
- 고객 응대 중심 서비스 제공
- 화장품 관련 정보 제공

---

## 🛠️ 기술 스택
### Backend
- **Spring Boot**
- **Oracle Database**
- **MyBatis**

### Frontend
- **Thymeleaf**
- **HTML / CSS / JavaScript**

### Tools & ETC
- IntelliJ IDEA / Eclipse
- Git & GitHub
- Maven

---

## 🎨 주요 특징
- **고객 중심**: 문의사항과 이벤트, 공지사항 등 고객 소통에 초점  
- **직접 디자인**: 메인 화면 및 전체 UI 디자인 직접 기획 및 제작  
- **안정성**: Spring Boot + MyBatis + Oracle을 활용한 안정적인 데이터 처리  
- **확장성**: 대댓글, 게시판 구조 등 다양한 확장 가능 기능 구현
